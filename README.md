계절학기 팀프로젝트로 진행한 "달릴래" 게임을 제작하였습니다. 

스피드 캐주얼 레이싱 게임으로 다함께차차차를 주제로하여 기획하였습니다. 

주인공은 일정한 금액을 모아야하는 스토리를 바탕으로 5개 라인의 도로에서 

차들을 피해 최대한 빠르게 배달을 해야되는 게임입니다.
